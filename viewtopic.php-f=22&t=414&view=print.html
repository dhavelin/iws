Irish Whiskey now on Whiskybase

Posted: Fri Feb 05, 2010 4:14 pm
by Willie JJ
It's been while coming but Menno Bachess has now added Irish Whiskey to the database at Whiskybase:

    Whiskybase

The site is a good fun way to see your collection and as it can be exported to Excel it can be useful for managing it. The number of Irish bottles in the database at the moment is quite small, but that can easily be changed by a bit of diligent effort from the users.

If there are labels that you have that are not in the database then you can pm them to Menno and he will add them. It can be a bit slow but they tend to get a lot of work when a new section is added. You can add bottles yourself as long as there is a section for it. They will check for info on the bottle before they confirm it, but it usually doesn't take too long.
