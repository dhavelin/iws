Re: Society Membership 2012

by Stephen » Thu Jan 05, 2012 1:15 am

Apologies for the delay, the codes have now been sent out to people with instructions on how to use them. I apologise if it seems complicated to use, but there is an element of manual checking required which will ensure that these are correctly used.

When using the codes please select the option Prepaid entry to the tasting event and enter the code you have been supplied with, this will discount the price 100%. Please ensure that you use the email address that the code has been sent to when booking entry to the event, this will make it easier to keep track of the use of the vouchers.

The reason why we are looking to close down the voucher bookings 1 week prior to the event is to see how much effort is involved in the administration of the vouchers, if this is not excessive then we will remove this condition.
