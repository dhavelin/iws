by DavidH
I find, like Mr IWC, that logo ideas are still popping into my head. I think there was some urgency to the logo hunt at one point but it's not so pressing any more, right? Let's see what emerges.

There is one other consideration for a logo design: scalability. If the logo can be described precisely in words in terms of basic shapes (rectangles, lines, circles...) and text then I can create a scalable vector graphic implementation myself and from there generate any imagery we need for the website and so on. That's great flexibility because a professional designer will probably supply the logo in eps format or something else we don't have the tools to manipulate.

If the image is complicated, creating a scalable graphic is a ton more work, must be done professionally and might be expensive. And it limits the possibility of adapting the logo in house.
