Open the other topic in your browser and copy the address from the URL box at the top. So for this topic, for example, the URL looks like this:
http://forum.irishwhiskeysociety.com/viewtopic.php?f=37&t=1413

There might be more in the URL but you just need this part. The 'f=37' describes which forum, the 't=1413' describes which thread.
