Not 100% sure G-W-H but initially it was a batch release thing and we have been told that tey release only a limited amount once a year.

The first 4 were from the following years

JJ18-1 (2002) Old Style Canister
JJ18-2 (2003)
JJ18-3 (2004)
JJ18-4 (2005)

Not sure if there was a JJ18-5 or 6 as I have never come across one but if anyone can confirm their existance it would be great. I suspect though if there is one released in 2006 one of the codes was dropped to even up the code with the year either from 2006 or 2007.
JJ18-5 (2006) or JJ18-6 (2006) but I reckon not both

JJ18-7 (2007)

JJ18-8 (2008) New Packaging Square card box first introduced in 2008

Lastly I think this coding has been dropped and is now just a static code for every year since.
I have not seen any further evidence of these codes since JJ18-8 and I have a JJ18-8 with the bottle code of L0 and a date of 26/03/10.

Again if anybody has evidence in regards to L9(2009), L1(2011) & L2(2012) just to confirm it would be great to hear. Check the for such codes and cross reference to the actual bottle code of JJ18-8.

Edit: Just to add not sure when it went from Master Selection to Limited Reserve but should be an easy one to figure out.

Pretty much the same thing ... However the 18yo is made/bottled in small batches with one batch being created every year. There can be differences between the batches but it is pretty much the same style of whiskey that they try to create every time. A bit like Midleton VR.

The positive side is that it had mainly improved on previous batches. Further I don't thing they are using a batch marking system any more. They may have removed the jj18 code altogether now but it stayed at jj18-8 for a long while even for different batches.
